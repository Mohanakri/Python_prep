## üî¢ What is an `int` in Python?

* `int` represents **whole numbers**, both positive and negative.
* No decimal points.
* Immutable and precise (no rounding issues like floats).

```python
x = 42
```

---

## üß© Group 1: Creation & Basic Operations

| **Operation / Method**   | **Description**                                             | **Example + Result**      |
| ------------------------ | ----------------------------------------------------------- | ------------------------- |
| `int(x)`                 | Convert x to integer                                        | `int("123")` ‚Üí `123`      |
| `int(x, base)`           | Convert from a given base                                   | `int("101", 2)` ‚Üí `5`     |
| `+`, `-`, `*`, `//`, `%` | Arithmetic: add, subtract, multiply, floor division, modulo | `5 % 2` ‚Üí `1`             |
| `**`                     | Exponentiation                                              | `2 ** 3` ‚Üí `8`            |
| `abs(x)`                 | Absolute value                                              | `abs(-7)` ‚Üí `7`           |
| `divmod(a, b)`           | Returns `(a // b, a % b)`                                   | `divmod(9, 4)` ‚Üí `(2, 1)` |

## üîé Group 2: Comparison & Boolean Operations

| **Operation**        | **Description**      | **Example + Result**               |
| -------------------- | -------------------- | ---------------------------------- |
| `==`, `!=`, `>`, `<` | Standard comparisons | `4 > 3` ‚Üí `True`                   |
| `and`, `or`, `not`   | Boolean logic        | `1 and 0` ‚Üí `0`, `not 1` ‚Üí `False` |
| `is`                 | Identity comparison  | `x is y` (compares memory)         |

## üîÅ Group 3: Bitwise Operations (for binary manipulation)

| **Operation** | **Description**         | **Example + Result**           |                    |         |
| ------------- | ----------------------- | ------------------------------ | ------------------ | ------- |
| `&`, \`       | `, `^\`                 | Bitwise AND, OR, XOR           | `5 & 3` ‚Üí `1`, \`5 | 3`‚Üí`7\` |
| `~`           | Bitwise NOT             | `~5` ‚Üí `-6`                    |                    |         |
| `<<`, `>>`    | Left shift, right shift | `2 << 1` ‚Üí `4`, `4 >> 1` ‚Üí `2` |                    |         |

## üßÆ Group 4: Utility & Conversions

| **Method / Function** | **Description**               | **Example + Result**           |
| --------------------- | ----------------------------- | ------------------------------ |
| `bin(x)`              | Convert to binary string      | `bin(5)` ‚Üí `'0b101'`           |
| `hex(x)`              | Convert to hexadecimal string | `hex(10)` ‚Üí `'0xa'`            |
| `oct(x)`              | Convert to octal string       | `oct(8)` ‚Üí `'0o10'`            |
| `round(x)`            | Round float to nearest int    | `round(3.7)` ‚Üí `4`             |
| `pow(x, y)`           | Power (same as `x ** y`)      | `pow(2, 4)` ‚Üí `16`             |
| `isinstance(x, int)`  | Check if object is integer    | `isinstance(10, int)` ‚Üí `True` |

## üß™ Example

```python
x = int("101", 2)     # 5
y = 3
print(x + y)          # 8
print(bin(x))         # 0b101
print(divmod(10, 3))  # (3, 1)
print(x & y)          # 1 (bitwise AND)
```
